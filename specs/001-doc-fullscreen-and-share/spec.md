# 文档全屏模式与单文档分享 需求规范

Version: 1.0
Created: 2026-06-18
Status: Draft

## 1. 概述

### 1.1 背景
当前知识库支持集合（Collection）级别的分享和文档查看，但：
- 文档查看区域受限于左右侧栏，长文档阅读体验不佳，缺乏专注阅读模式
- 分享粒度仅到集合，无法单独分享某个文档给团队协作

### 1.2 目标
- 提供文档全屏专注阅读模式，提升长文档阅读体验
- 支持单个文档级别的只读分享，满足团队内部协作场景

### 1.3 范围
MVP：
- 文档全屏模式（Markdown/HTML 均支持）
- 单文档分享：生成/复制/撤销分享链接

后续迭代：
- 已分享状态标识、顶栏菜单入口、分享页元信息展示、链接过期、可下载

### 1.4 成功指标
- 全屏模式被用户用于阅读长文档
- 单文档分享功能被使用，减少集合整体分享的需求

## 2. 用户分析

### 2.1 目标用户
- **知识库管理员**（已登录）：管理文档、生成分享链接、使用全屏阅读
- **团队成员**（无需登录）：通过分享链接只读查看单个文档

### 2.2 用户故事

**作为** 知识库管理员
**我想要** 在阅读长文档时进入全屏模式
**以便于** 专注阅读，不受侧栏干扰

**作为** 知识库管理员
**我想要** 分享单个文档的只读链接
**以便于** 团队成员无需登录即可查看特定文档

**作为** 团队成员
**我想要** 通过分享链接查看文档
**以便于** 快速获取所需文档内容

### 2.3 用户旅程

**全屏阅读：**
1. 管理员打开文档查看
2. 点击全屏按钮
3. 文档内容全屏展示（精简标题栏 + 内容区）
4. 阅读完毕，ESC 或退出按钮退出

**文档分享：**
1. 管理员在文档列表 hover 文档项，点击分享按钮
2. 系统生成分享链接，弹窗展示
3. 管理员复制链接，发给团队成员
4. 团队成员打开链接，只读查看文档
5. 管理员可随时撤销分享

## 3. 功能需求

### 3.1 核心功能
| 功能点 | 描述 | 优先级 | 验收标准 |
|--------|------|--------|----------|
| 文档全屏模式 | 文档内容全屏展示，精简标题栏 | P0 | Markdown/HTML均可全屏，ESC退出 |
| 全屏退出 | ESC键或退出按钮 | P0 | 两种方式均可退出 |
| 生成文档分享链接 | 为单个文档生成只读分享链接 | P0 | 生成唯一token，返回链接 |
| 复制分享链接 | 复制链接到剪贴板 | P0 | 复制成功有反馈 |
| 撤销文档分享 | 撤销分享，链接失效 | P0 | 撤销后链接404 |
| 访问分享文档 | 无需登录只读查看 | P0 | 通过链接查看文档内容 |

### 3.2 功能详细说明

#### 功能A：文档全屏模式
- **触发条件**：管理员在文档查看页，点击全屏按钮
- **操作流程**：点击全屏 → 覆盖层展示（精简标题栏：文档标题+退出按钮；内容区：文档内容）→ ESC/按钮退出
- **业务规则**：Markdown内容可滚动；HTML内容正确渲染；全屏时隐藏左右侧栏
- **输出结果**：沉浸式阅读视图

#### 功能B：生成文档分享链接
- **触发条件**：管理员在文档列表 hover 文档项，点击分享按钮
- **操作流程**：点击分享 → 生成链接 → 弹窗展示链接 + 复制按钮 → 复制/关闭
- **业务规则**：每个文档一个分享token；链接永久有效直至撤销；只读访问
- **输出结果**：可分享的只读链接

#### 功能C：撤销分享
- **触发条件**：管理员撤销某文档分享
- **操作流程**：点击撤销 → token失效 → 链接访问404
- **业务规则**：撤销后链接立即失效

#### 功能D：分享页访问
- **触发条件**：访问者打开分享链接
- **操作流程**：打开链接 → 加载文档内容 → 只读展示
- **业务规则**：无需登录；只读；仅展示该文档

### 3.3 交互说明
- 全屏按钮位于文档顶栏操作区
- 分享按钮位于文档列表项，hover 显示
- 分享弹窗：链接输入框（只读）+ 复制按钮 + 关闭按钮
- 复制成功/失败有明确提示

## 4. 数据需求

### 4.1 数据展示
- 全屏：文档标题、文档内容（Markdown渲染/HTML渲染）
- 分享弹窗：分享链接URL
- 分享页：文档内容

### 4.2 数据操作
- 生成分享token（创建）
- 撤销分享token（删除/置空）
- 查询分享文档（通过token）

### 4.3 数据校验规则
- 分享token唯一
- 文档删除时分享自动失效

## 5. 非功能需求

### 5.1 性能要求
- 全屏切换无明显卡顿
- 分享页加载与正常查看相当

### 5.2 兼容性要求
- 现代浏览器（Chrome/Edge/Firefox）
- 响应式适配

### 5.3 安全要求
- 分享链接仅限只读访问单个文档
- 生成/撤销分享需鉴权（仅管理员）
- 分享不泄露集合内其他文档
- HTML分享内容经净化（防XSS）

## 6. 边缘情况与异常处理

| 场景 | 处理方式 |
|------|----------|
| 无效/已撤销分享链接 | 返回404，提示链接无效 |
| 原文档被删除 | 分享自动失效，访问404 |
| 分享页加载失败 | 显示错误提示 |
| 全屏时切换文档 | 退出全屏或更新内容（MVP：退出全屏） |
| 复制链接失败（环境限制） | fallback机制 + 明确提示 |

## 7. 验收标准

### 7.1 功能验收
- [ ] 文档全屏：Markdown/HTML均可全屏，精简标题栏，ESC/按钮退出
- [ ] 生成分享：点击分享生成链接，弹窗展示
- [ ] 复制链接：复制成功有反馈
- [ ] 撤销分享：撤销后链接404
- [ ] 分享访问：无需登录只读查看文档
- [ ] 删除联动：文档删除后分享失效

### 7.2 体验验收
- [ ] 全屏切换流畅
- [ ] 分享按钮hover显示，不干扰列表项点击
- [ ] 提示信息清晰
- [ ] 分享页阅读体验良好
